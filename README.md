# FPGA Accelerator RTL Lab

A collection of FPGA accelerator building blocks implemented in Verilog, progressing from basic compute units to complete FPGA accelerator architectures.

## Objectives

* RTL Design
* FPGA Design
* Digital Design
* AI Accelerator Development
* Verification Methodologies
* SoC Integration

## Repository Structure

```text
rtl/
├── mac/
├── pe/
├── systolic_array/
├── matrix_multiplier/
└── axi/

tb/
├── mac_tb/
├── pe_tb/
├── systolic_tb/
└── regression/

fpga/
└── zedboard/

verification/
diary/
reports/
```
